# YouTube Quick Unsubscribe

A [Violentmonkey](https://violentmonkey.github.io/) userscript that adds a quick **Unsubscribe** button under each video on YouTube, and an **Unsubscribe** option in the ⋮ menu.

## Install

1. Install [Violentmonkey](https://violentmonkey.github.io/) for your browser.
2. Create a new script in Violentmonkey and paste in the script contents.

## Features

- **Unsubscribe button** beneath every video card on your feed
- **⋮ menu item** — "Unsubscribe from \<channel\>" in the video context menu
- Works with YouTube's new lockup layout
- No external dependencies
